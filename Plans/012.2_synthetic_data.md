## Goal

Generate synthetic price data using Monte Carlo simulation to test strategies beyond historical data. This validates whether a strategy's edge is statistically robust or merely a product of curve-fitting to specific market conditions. The same backtest engine from Phase 10 is reused, but fed synthetic candles instead of real ones.

## Approved Design (Brainstorm 2026-03-28)

- **12.2.1 — Internalize MC package:** Copy the Monte Carlo engine from ops-worker into IRT codebase. Self-contained, no external dependency on ops-worker repo.
- **12.2.2 — Backend foundation:** New DB columns/migration for montecarlo mode (`mode` enum: historical | montecarlo), `n_paths` (int), `fit_years` (float). API endpoints accept MC params alongside existing backtest request.
- **12.2.3 — Worker MC mode:** The unified worker (from 12.1) gains a `montecarlo` mode. Orchestrator decomposes MC run into subprocess invocations. Timeframe remap maps IRT timeframes to MC engine expectations.
- **12.2.4 — Frontend MC UI:** Third mode button in backtest panel (Historical | Multi-TF | Monte Carlo). Config form for n_paths + fit_years. TypeScript types for MC request/response.
- **12.2.5 — Frontend MC report:** Fan chart (price paths), histogram (final PnL distribution), percentile table (p5/p25/p50/p75/p95), risk table (max drawdown distribution), overfitting card (historical vs synthetic comparison).
- **12.2.6 — Verification:** End-to-end MC run with a known strategy, validate report rendering, edge cases (single path, zero fit_years, missing data).

## Notes

- Geometric Brownian Motion (GBM) is the standard starting point for synthetic price generation
- Should support configurable parameters: drift, volatility, number of paths, time horizon
- Each Monte Carlo run produces N synthetic price series; the strategy is backtested on each
- Results are stored as a distribution (not a single outcome) for statistical analysis
- Consider seeding for reproducibility of specific runs
- The worker from Phase 12.1 is reused — this phase extends it with MC mode, not replaces it
