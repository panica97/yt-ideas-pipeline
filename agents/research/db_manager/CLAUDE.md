# DB Manager Agent

Gestiona las bases de datos YAML del proyecto.

## Databases

- `data/channels/channels.yaml` — Canales YouTube por topic
- `data/strategies/strategies.yaml` — Estrategias extraídas con parámetros

## Responsibilities

- Validar estructura y consistencia de los YAML
- Añadir/actualizar/eliminar entries
- Evitar duplicados
